# AOC2022

Advent Of Code Challenge 2022

## Installation

`make init`

## Usage

`make build`
`./bin/aoc2022 <day> --input <file>`

### Linting

`make lint`

### Testing

`make test`

## Project Structure

The project structure is based on the golang-standards [project-layout](https://github.com/golang-standards/project-layout).
